# FSO-blog-app

This repo contains front- and backend of application made during Mooc FSO course (parts 0-7): https://fullstackopen.com/

Key features and things learnt/used during the project:
- React states, redux
- SPA (React router)
- React bootstrap
- Express
- User authentication
